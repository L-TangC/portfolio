Concept

製作這個專案對我而言是個漫長的旅途，從零到有的去自學UI/UX的技能以及HTML、CSS、JavaScript，也是由我自己一刀一比刻出來的網站。 這個作品集網站不只是我展示作品的平台，更是我給自己學習Coding與UIUX後的一個挑戰，更完整的紀錄從頭到尾的過程。

Ideate

在製作這個網站之前，其實有做一個舊的作品集網頁的mockup，但是隨著學到的技能越來越多，便決定重新做一個網站。
我在練習Coding的過程中，突然萌生的想法，挑戰自己看自己是否有真的能把學到的技能融入自己的作品與想法之中。

Design

Style

在風格上想做出讀出一格的風格，讓造訪者可以能對作品印象深刻，因此我製作了一個突破畫面平衡卻又不失整體觀看體驗的斜角。 在顏色使用上做出較有視覺落差的淺粉紅漸層與灰色為主色調，帶有活潑卻不失莊重。

Color

Interaction

首先使用了斜角的頁首與頁尾，不只做出特色，更是一種優化menu的點擊的方式，因為在選單內容不多的情況下，放成直式選單在畫面邊角，可以直接用大拇指做選取，更加直覺。
網頁中運用了視差滾動的效果和更用運CSS的一些效果，做出作品放大預覽的功能來呈現出一些特殊性。 而在聯絡我的部分運用彈跳視窗的方式呈現，瀏覽者可以隨時隨地的查看我的資訊方式。
最後更是做了中英文可切換的版本提供不同語言的使用者造訪網站。

Mockup

Coding

HTML

案中最為困難的部分是斜角的製作，因此在HTML裡Class的分層做對，才能有辦法在CSS中去執行而不會出現無法適應背景色的情況。

CSS

寫CSS的過程因要去執行大量的分類與分層，因此需要去整理CSS的打法，盡量去簡化寫法，在現有的經驗下讓Coding的畫面比較乾淨，未來修改上也會比較方便。 精簡CSS後去製作RWD自適應上也比較輕鬆，不會需要去覆蓋太多原先的檔案。

JavaScript

我在JS上的執行相較比較少，只有用在「聯絡我」彈跳視窗中運用，另外也讓CSS去完成大多的效果。
